fsMTS

Implementation of feature selection methods for multivariate time series

News

  • First version is released (2020-March)

Supported feature selection algorithms:

  • Own lags implements selection of own lags for every component of a multivariate time series
  • Distance-based implements feature selection based on an external structure of distances between components of a multivariate time series
  • Cross-correlation-based utilises Pearson's cross-correlation coefficients for identification of relationships between components of a multivariate time series
  • Graphical LASSO-based utilises the graphical LASSO algorithm for finding a sparse representation of the multivariate time series' covariance matrix
  • Random forest-based utilises the random forest algorithm for identification of relationships between components of a multivariate time series
  • Least angle regression-based utilises the least angle regression for identification of relationships between components of a multivariate time series
  • Mutual information-based utilises mutual information for identification of relationships between components of a multivariate time series

Repositories

  • The released version is available on CRAN
    • You can download the source code and binaries from http://cran.r-project.org/web/packages/fsMTS/index.html.
    • You can install the package by typing at the R console: 
       install.packages( "fsMTS" )
  • The development version is available on R-Forge
    • You can download the source code and binaries from https://r-forge.r-project.org/scm/?group_id=2392.
    • You can install the package by typing at the R console: 
       install.packages( "fsMTS", repos="http://R-Forge.R-project.org" )
